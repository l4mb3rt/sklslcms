# sklslcms
Suicide Kings Loot System Light CMS

A simple CMS to manage Groups and Raids with the Suicide Kings Loot System mainly for WoW.

Planned Features:
- Add user as admin or group admin
- Add groups
- Add lists
- Add charackters
- Manage Loot lists
- Add items
- Display loot history for a single group
.
.
.
